# 生词

**schema：**

1. 纲要、图解
2. 在数据库领域，schema 是指数据库的结构定义，描述了数据库中表、字段、关系、约束等内容的组织方式
3. 不同 schema 含义可能不同、核心概念始终围绕着结构化描述 和规则定义
